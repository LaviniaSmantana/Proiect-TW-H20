1. Overview

We created this web application to help you manage your preferences for non-alcoholic beverage consumption. We also offer support for the management of related data: category, price, ingredients, images, restrictions, validity and others. Our system will provide support for shopping list activities, plus user administration.

2. Landing Page

The first page of this website provides the name and the option to login or register.

3. Register

Registering can be done very easily by providing an username, email, password and a picture, if the user decides to do so. After doing this, the account will be saved as a normal user one and he/she will be redirected to the Login Page.

4. Login

If the user has already created an account, the only thing the user has to do is to type in the username and the password correspondent to, and he/she will be redirected to the Home Page. Also, if he/she doesn't have an account there is the option to go to the register page by clicking the
'Regiser here' link.

5. Home Page

This is the page where it all begins, presenting the web application with a pretty design, but its most important element is the navigation bar. To find out more about how to use it, go to the Navigation Bar section below. The user will be able to see this week's top 3 most selling drinks that are available as RSS data flow, alongside with the other products that are displayed in a decreasing manner. Also, the home page shows all of the administrators contact details.

6. Navigation Bar

The purpose of the navigation bar is to help the user explore the website easily. Thus, there are four choices the user can make when it comes to the navigation bar, explained in the following sections.

6.1. Your Account

In this page, the user has the options of changing his profile.

  • His/her username.
  • His/her profile picture.
  • His/her password.

6.2. Products

To find out more about how to use it, go to the Navigation Bar section below.

By accessing this link, the user will be directed to the websites product catalog. Here, the initial nav bar will be slightly changed; instead of the 'Products' link, there will
be a 'Home' link provided, which will send the user back to the Home page.
The main purpose of this page is to display all of the categories available:

  • Carbonated drinks
  • Noncarbonated drinks
  • Water
  • Energy Drinks
  • Syrups
  • Milk

6.3. Statistics

In this page, the user will be able to see two statistics: one that describes the amount of
sugar in some of the drinks, and the other that shows the percentage of the quality of the drinks. There is the also the option to export the chart statistic as SVG file.

'

6.4. Log out

By clicking this, the user will be redirected to the login page.

7. Categories of Drinks

Once the user has selected the category from the catalog page, he/she will pe directed to that specific category page, that displays all the beverages available. In this page, the nav bar is again slightly changed: the link for the statistics is replaced by the shopping cart link. All of the products can be accessed by clicking on the picture and have the option of adding it to the shopping cart by pressing the 'Add to cart' button. To find out more about the single product page, go to the Single Product Page section below.

8. Single Product Page

This page will provide the user several details about the chosen product:

  • Name
  • Price
  • Ingredients, Category, Origin of Ingredients, Manufacture or processing place
  • Nutri and Nova Score.
  • Table with nutritional facts that can be exported as CSV file.

9. Shopping Cart

Another feature the user can enjoy is the posibility of adding a product to the shopping cart. The quantity of a product can be increased up to five, decreased or fully removed. The price will be updated no matter the action.

10. User

The functionalities mentioned above are available for any user.
