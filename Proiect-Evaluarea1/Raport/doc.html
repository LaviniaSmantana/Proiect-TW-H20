1. Web design

We started the project by creating the main elements such as the main page, the login and register pages, a page that displays the beverages and a page with the user's information. Each beverage will have a individual page where it will show nutrition details about the drink.

2. Login and Register

Right now we have the login and register pages, such that if somebody wants to enter the webpage he/she needs to create an account firstly.

3. Responsiveness

We have modified the width and the height values so that for now, the main page can also be properly seen on a tablet or smartphone.

4. Products

Right now we have one category of drinks that are displayed on one page. Each drink has a different photo, name and price.

5. Administration

For keeping the code updated and managing it we used Trello and Github. Every time one of us had some useful update for the project, we would post on Trello and then commit and push it.
